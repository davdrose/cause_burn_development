# R Script

Data analyses and figures for the paper were generated using the script in `main.Rmd`. `cache` contains the bayesian models that were run to analyze the data. `order_effects.Rmd` and `rsa.Rmd` contains analyses data analyses and figures included in the supplemental information file.

To reproduce all analyses and figures, simply fork the repository, open `cause_burn_dev.Rproj` in RStudio, load `main.Rmd` (or `order_effects.Rmd` or `rsa.Rmd`) and click `run all`.
